Rules for naming JS variables



Variable names can only contain letters, numbers, $ sign and underscores.For example $my_1stVariable.

Variables must begin with a name, numbers ounderscore. For example $name, _name or name.

Variable names are case sensitive.

Variable names should not be JS keyword.
